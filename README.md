# 概要
 - 上下が平たいタイプのへクスマップエディターです。
 - TRPG用のマップ作製ツールとして開発しましたが、他にも役立つかもしれません

# 機能
## ツール 
 - タイルごとに色のついた枠付きマスを作成/削除ができます。
 - プリセット機能で作成したタイル設定を保存し、再利用が可能です。
 - ミドルボタンで選択したタイルの設定をコピーし、作業を効率化します。
 - マスごとに数値IDを設定でき、識別がスムーズになります。

## 設定
 - ID設定: IDの表示非表示、色配置などを設定できます。
 - ボーダー設定: ボーダーの表示非表示、色などを設定できます。
 - タイル設定: 盤上のタイル数を調整できます。
 - 余白設定: タイルごとの距離を調整できます。
 - 透明度: 画像出力時の透明度を調整できます。

## インポート/エクスポート
 - 画像出力: 作成したマップを画像として出力します。
 - json出力: マップや設定含めて、jsonファイルとして出力します。
